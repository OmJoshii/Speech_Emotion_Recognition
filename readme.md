# Speech Emotion Recognition System

A deep learning system that detects human emotions (happy, sad, angry, neutral, etc.)
from speech audio using CNN and LSTM models.

## Dataset
Download RAVDESS dataset from:
https://www.kaggle.com/datasets/uwrfkaggler/ravdess-emotional-speech-audio
Extract and place inside the `data/ravdess/` folder.
- 2880 speech audio files
- 8 emotions: neutral, calm, happy, sad, angry, fearful, disgust, surprised
- 24 actors (12 male, 12 female)
